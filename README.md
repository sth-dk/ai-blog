# ChuangLian's AI Blog

个人 AI 技术博客，记录 AI 领域的探索、学习和思考。

> 规避痛苦，好好爱自己。

**在线访问：** https://sth-dk.github.io/ai-blog/

## 写新文章

在 `src/content/blog/` 目录下新建 `.md` 文件：

```markdown
---
title: '文章标题'
description: '一句话描述'
pubDate: 2026-04-21
tags: ['标签1', '标签2']
---

正文内容...
```

## 发布

```bash
git add -A
git commit -m "文章标题"
git push
```

推送后自动部署，约 1-2 分钟上线。
